## All the zen BOT

### Usage
```sh
$ node index.js --help
Usage: azen [options]

All the zen BOT

Options:
  -V, --version              output the version number
  -p, --port <port>          Specify the TCP port on which the server is listening for connections.
  --x-id-token <x-id-token>  Specify the AllTheZen token on which server is using for authentication.
  -h, --help                 display help for command
```
