## All the zen BOT

### Usage
```sh
$ node allthezen.js --help
Usage: azen [options]

All the zen BOT

Options:
  -V, --version                              output the version number
  --allow-listen                             Specify the server status on which server is using for listening.
  -p, --port <port>                          Specify the TCP port on which the server is listening for connections.
  --ignore-proxy                             Specify the proxy status on which server is using for connections.
  --target-cat-category <targetCatCategory>  Specify the target cat category on which server is using for auto
                                             upgrade
  -h, --help                                 display help for command

```
